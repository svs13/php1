п.3
Из мануала:
>Скобки не обязательны. Будьте внимательны при сравнении возвращаемого значения.
>if (include('vars.php') == TRUE) { ... и if ((include 'vars.php') == TRUE) { ...
При успешном включении файла оператор include возвращает int(1).
При неудачном включении файла оператор include возвращает bool(false) и выводит в поток предупреждение.
Если выполнение включаемого файла закончено оператором return, то оператор include возвращает переданное значение return

п.4
Угадать пол по имени.
Первый вариант - перебор имён. В итоге - точное определение пола, но огромное разнообразие имён. Это не есть угадывание.
Второй вариант - перебор по последней букве имени.
Женские имена(полные) https://ru.wikipedia.org/wiki/Категория:Русские_женские_имена
Анна, Мария, Наталья, Елена, Алёна
Мужские имена(полные) https://ru.wikipedia.org/wiki/Категория:Русские_мужские_имена
Иван, Дмитрий, Владимир, Альберт

Условие определения пола - окончание имени:
- женские: а, я
- мужские: й,н,р,т,м,х,м,п,в,д,л
null: остальные.
